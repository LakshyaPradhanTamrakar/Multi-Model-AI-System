
Deep Learning Basics: Covered how neural networks use neurons, layers, weights, and biases. Looked at how forward propagation and backpropagation work, how loss functions and gradient descent help the network learn, and why activation functions (like ReLU, Sigmoid, and Softmax) introduce necessary non-linearity.

CNN Mechanics: Learned why CNNs excel at handling image data compared to standard networks. Explored convolution layers (using filters and kernels to create feature maps), max pooling for downsampling, and how flattening leads into fully connected layers for tasks like image classification and object detection.

Intro to LLMs: Explored Large Language Models, looking at how they train on massive text data. Learned how tokenization breaks text down into tokens, how the Transformer architecture uses self-attention to track context, and what makes GPT-style models so good at generating text.
